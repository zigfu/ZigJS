Zigfu's ZigJS Beta

ZigJS allows anyone to write gesture driven web pages, as well as full body motion games in html5/unity3d webplayer/flash.
Please share your experiences with us at beta@zigfu.com

Install

  1. Install latest version of OpenNI: Windows | Mac
  2. Install the browser plugin: Windows | Mac
  3. Important: Restart your browser

Make sure it works

Once everything is installed and the browser restarted, the plugin version and users radar should work as expected
